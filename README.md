AI Voice Calling Agent Platform
Introduction

This project is an AI-based voice calling platform that supports both inbound and outbound calls. The main goal of the project is to help businesses automate customer communication using AI voice agents while keeping the cost very low compared to other platforms available in the market.

The system is designed mainly for Indian businesses and supports Indian phone numbers through Vobiz.ai
. The complete backend and data storage are handled using Supabase
, while LiveKit
 is used for real-time voice communication and text-to-speech functionality.

One of the most interesting parts of this project is its pricing. The platform costs only around ₹1.25 per minute, which is much cheaper when compared with platforms like Vapi
, Retell AI
, and Omnidim
.

Project Features
Inbound AI Agent

The inbound AI agent handles incoming customer calls automatically. Whenever a customer calls the business number, the AI agent answers the call and communicates with the customer in real time.

The AI agent can:

Answer customer questions
Provide information about services
Collect customer details
Handle support queries
Store the complete conversation

All call data and conversation history are saved in the backend database for future reference.

Outbound AI Agent

The outbound AI agent is used to make automated calls to customers or leads. The admin can upload leads through the dashboard, and the AI agent will automatically start calling them.

The outbound system can be used for:

Lead generation
Follow-up calls
Appointment reminders
Marketing campaigns
Customer feedback collection

After every call, the system generates a summary automatically.

Admin Dashboard

The platform includes an admin dashboard where the admin can monitor and manage the entire system.

The dashboard allows the admin to:

View all leads
Check call history
Monitor conversations
Read call transcripts
Track call status
Analyze customer responses

The admin can also see which calls were successful and which leads require follow-up.

Backend and Data Storage

The backend of the project is built using Supabase
. It stores all important information related to calls and customers.

The backend stores:

Lead information
Call transcripts
Call duration
Customer responses
AI-generated summaries
Call records

This makes it easy to manage and analyze all conversations from one place.

Real-Time Voice Communication

The project uses LiveKit
 for text-to-speech and real-time audio streaming.

LiveKit helps in:

Real-time communication
Fast voice response
Smooth audio streaming
Human-like AI conversations

This improves the overall user experience and makes conversations feel more natural.

Telegram Bot Integration

One of the useful features of this platform is Telegram bot integration.

After every call:

The conversation is processed
A summary is generated by AI
The summary is sent directly to the Telegram bot

The Telegram notification includes:

Customer name
Phone number
Call summary
Lead status
Important conversation points

This helps the admin receive updates instantly without opening the dashboard every time.

Cost Efficiency

Most AI calling platforms are expensive and difficult for startups or small businesses to afford. This project focuses heavily on reducing operational costs.

The platform operates at approximately ₹1.25 per minute, which is significantly cheaper than many existing AI calling solutions in the market.

Compared to:

Vapi
Retell AI
Omnidim

this platform provides a much more affordable solution while still maintaining good performance and scalability.

System Workflow
Outbound Call Workflow
Admin uploads lead data
AI agent starts outbound calling
Customer answers the call
AI communicates with the customer
Conversation is stored in backend
AI generates a summary
Telegram notification is sent to admin
Inbound Call Workflow
Customer calls business number
AI agent receives the call
AI communicates with customer
Entire conversation is stored
AI summary is generated
Admin reviews call details in dashboard
